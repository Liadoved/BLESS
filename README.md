# BLESS Forms Frontend

טופס העלאת תמונות לארכיון BLESS, בנוי עם Next.js ו-Tailwind CSS.

## תכונות
- טופס מודרני ונוח לשימוש
- תמיכה מלאה בעברית
- העלאת תמונות בגרירה
- עיצוב רספונסיבי
- אנימציות חלקות

## טכנולוגיות
- Next.js 14
- TypeScript
- Tailwind CSS
- Heroicons

## התקנה

```bash
# התקנת חבילות
npm install

# הרצת שרת פיתוח
npm run dev
```

הדף יהיה זמין ב-http://localhost:3000

## מבנה הפרויקט
- `src/app/page.tsx` - דף הבית עם הטופס
- `src/app/layout.tsx` - תבנית הבסיס של האתר
- `src/app/globals.css` - סגנונות גלובליים
- `tailwind.config.ts` - הגדרות Tailwind

## תודות
- גופן Assistant מ-Google Fonts
- אייקונים מ-Heroicons
